  Стек технологий:
  Frontend: React, Redux-tookit, axios
  Backend: Node.js + Express  
  Database: MySQL  

 
**Гость:** имеет возможность зарегистрироваться и авторизоваться, просмотреть главную страницу, посмотреть букеты в наличии в различных категориях: букеты на свадьбу, букеты на день рождения и т.д.  
**Авторизованный пользователь:** имеет дополненный функционал гостя с возможностью добавлять заказы в корзину, отменять или оплачивать их, а так же просматривать свои заказы.  
**Администратор:** имеет возможность редактирования категорий букетов и самих букетов и добавления новых, просмотра заказов пользователей и изменения их статуса, фильтрации заказов по статусу, дате и пользователю.  
  
  
### Запуск приложения    
1. ```console
   cd frontend   
   npm start
   ```

       
2. ```console
   cd backend    
   npm start
   ```
 
 ### Запуск панели администратора    
```console
  cd backend    
  cd npm run mysql-admin
